# יד שרה — מערכת ניהול רפואה דחופה

מערכת לניהול מסע המטופל במחלקת הרפואה הדחופה של יד שרה — מרגע ההגעה לקבלה, דרך התור
והטיפול, ועד לשחרור. מחליפה את המערכת הקיימת.

## עקרונות-על

- 🔒 **אבטחה** בתקני ארגון בריאות מחמירים (PHI).
- 🌐 **אי-תלות באינטרנט** — המערכת חיה כולה ברשת פנימית (On-Prem LAN).
- ✍️ **עריכה מרובת-משתמשים** של אותו טופס עם מניעת דריסה בו-זמנית.
- 🔁 **התממשקות** (interoperability) וייצוא נתוני טיפול בסוף הטיפול.

## Stack

React + TypeScript · ASP.NET Core · PostgreSQL · SignalR — הכול On-Prem, API-first.
